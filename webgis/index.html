Metodologi Data & Proses

  • Dataset: Sentinel-2 Harmonized (Level-2A).
  • Rentang Citra: Periode 1 (Juli 2023 - Des 2024), Periode 2 (Jan - Des 2026).
  • Fitur/Indeks: NDVI, NDBI, B2, B3, B4, B8, B11, B12.
  • Algoritma: Random Forest Classifier (100 Trees).
  • Ground Truth: 400 titik terbagi seimbang (100 Target, 100 Non-Target per periode).
  • Validation: Train 80% / Test 20% split dengan Seed statis (42).

Evaluasi Model (Random Forest)

Model dievaluasi menggunakan data testing (20%) yang dipisahkan secara acak (Seed: 42). Hasil menunjukkan kinerja sebagai berikut:

Metrik Class 0 (Non-Target) Class 1 (Target/Built-up)
Producers Accuracy (Recall) 0.95 0.92
Consumers Accuracy (Precision) 0.94 0.96
F1-Score 0.94 0.93

*Note: Nilai dapat berbeda sesuai hasil run GEE terbaru.

Insight Perubahan Spasial

Dari hasil deteksi perubahan, ditemukan pola dominan:

  • Penambahan Area Terbangun (Gain): Terpusat di KIPP (Kawasan Inti Pusat Pemerintahan) dan jalur infrastruktur utama seperti jalan tol Balikpapan-IKN.
  • Pengurangan Area Terbangun (Loss): Relatif kecil, seringkali merupakan hasil dari reklamasi area proyek sementara.
