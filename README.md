# IKN Built-Up Area Change Detection (2023/2024 vs 2025)

Proyek ini adalah implementasi WebGIS dan Machine Learning (Random Forest) menggunakan Google Earth Engine (GEE) untuk mendeteksi perubahan area terbangun di Ibu Kota Nusantara (IKN).

## 📁 Struktur Direktori
* `gee/` - Script JavaScript untuk Google Earth Engine
* `data/` - File GeoJSON hasil klasifikasi dan titik sampel
* `webgis/` - File HTML, CSS, dan JS untuk peta interaktif
* `results/` - Metrik evaluasi (Confusion Matrix, APRF)
* `report/` - Laporan akhir PDF dan slide presentasi

## 🚀 Cara Penggunaan
1. Jalankan script di folder `gee/` menggunakan [GEE Code Editor](https://code.earthengine.google.com/) secara berurutan.
2. Download hasil export (GeoJSON) dari Google Drive ke folder `data/`.
3. Buka `webgis/index.html` di browser untuk melihat peta interaktif.
